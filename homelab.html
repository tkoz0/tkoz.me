Homelab

This page describes my janky homelab setup. Someone has to pay the power bill for this thing... (around $100/month)

Everything is connected to a cheap 8 port gigabit switch which is connected to a router. I have everything on a wooden rack I designed and built myself. The equipment is attached to an APC Smart UPS SUA1500RM2U.

Rimuru (main server)

Named after the main character in the anime Tensei Shitara Slime Datta Ken (Tensura). I use this one for data storage and running various services.

  • Dell PowerEdge R720xd
  • 1x Intel Xeon E5-2643 v2 (6C/12T 3.5GHz)
  • 64GB (2x32GB) PC3-10600R memory
  • 12x8TB SAS HDD in ZFS RAIDZ2
  • 1TB PCIe NVMe SSD for OS
  • 2x750W Power Supplies
  • Ubuntu Server 22.04

Lacia (compute server)

Named after a character in the anime Beatless. I use this one for running heavy computing projects including my own math code for research.

  • Dell PowerEdge R720
  • 2x Intel Xeon E5-2697 v2 (24C/48T 2.7GHz)
  • NVidia Tesla P100 16GB GPU
  • 512GB (16x32GB) PC3L-10600R memory
  • 8x4TB SATA HDD
  • 1TB PCIe NVMe SSD for OS
  • 2x750W Power Supplies
  • Ubuntu Server 22.04
